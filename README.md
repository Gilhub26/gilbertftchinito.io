# ✨ Professional CV Builder

A clean and modern web-based CV/Resume builder using **HTML, CSS, and PHP**.

Built with:
- PHP Superglobals (`$_POST` and `$_FILES`)
- Beautiful modern UI with gradient design
- Profile picture upload support
- One-click Print / Save as PDF

## 🚀 Features

- Responsive and elegant design
- Real-time CV generation
- Professional layout with left & right columns
- Easy to customize
- Works perfectly with XAMPP / WAMP

## 🛠️ Technologies Used

- **Frontend**: HTML5, CSS3 (Flexbox + Grid)
- **Backend**: PHP 8
- **File Handling**: `$_FILES` for profile picture upload
